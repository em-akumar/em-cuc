REO Dropdown For Tablet


REO dropdown control usage
              <div class="demo-example">
    <label>Dropdown Large</label>
    <div class="em-reo-dropdown-t" options="vm.drpCountry" emc="ReoDropdown"></div>
      <br/>
      <label>Dropdown Medium</label>
      <div class="em-reo-dropdown-t" options="vm.drpCountry2" emc="ReoDropdown"></div>
    </div>

 this.drpCountry = {
      defaultText: 'Select',
      onChange: (e) => {},
      valueField: 'text',
      textField: 'text',
      defaultSize: 'large',
      itemList: [{ 'value': 1, 'text': 'State1', 'class': 'disabled' }, { 'value': 2, 'text': 'State2' }]
    };
Class/Properties Description
.em-reo-dropdown-t This class is required. Indicates reo dropdown
defaultText Display default selected text on the dropdown
onChange Expose OnChange method, It helps to change the items on the selection
valueField The value which user wants to keep or store in the database
textField Display text on the selection of dropdown
defaultSize Define the size of drodown
