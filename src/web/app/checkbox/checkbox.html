                Class
                Description
            

            

                .checkbox
                Checkbox are created with the .checkbox class
            

            

                .checkbox-group-header
                Adds a heading to the checkbox group
            

            

                .em-readonly-checkbox
                Set readonly state of the checkbox.Add onclick='return false' event to readonly state of checkbox
            

            

                .em-readonly-checked-checkbox
                Set readonly and checked state of the checkbox.Add onclick="return false" event to readonly and checked state of checkbox
            

            

                .em-restricked-checkbox
                Set readonly and checked state of the checkbox.Convert checkbox label text into dots
            

            

                .em-errorCheckbox
                Display the error state in red color
            

            

    


    
    

        

            

                Property
                Description
            

            

                checked
                Adds a checked state of the checkbox
            

            

                disabled
                Adds a disabled state of the checkbox
