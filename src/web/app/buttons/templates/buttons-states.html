Button State Usage
<div class="demo-example" data-heading="Primary text button states">
     <button type="button" class="btn btn-primary">Default</button>
     <button type="button" class="btn btn-primary">Hover</button>
     <button type="button" class="btn btn-primary active">Active</button>
     <button type="button" class="btn btn-primary disabled" data-toggle="tooltip" title="Additional info on the action" >Disabled</button>
</div>

    

  


    
    

        

            

                Class
                Description
            

            

                .active
                Makes the button appear pressed
            

            

                .disabled
                Makes the button disabled
            

            

    


    
    

        

            

                Event
                Description
            

            

                data-toggle="tooltip"
                Add tooltip to the button
            

            

                title
                Add button tooltip description
