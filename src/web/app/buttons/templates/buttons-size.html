Button Sizes Usage
<div class="demo-example" data-heading="Samples (Fixed Width)">
     <button type="button" class="btn btn-primary btn-xs">OK</button>
     <button type="button" class="btn btn-primary btn-sm">Small</button>
     <button type="button" class="btn btn-primary btn-md">Medium</button>
     <button type="button" class="btn btn-primary btn-lg">Large</button>
     <button type="button" class="btn btn-primary btn-block">Flexible</button>
</div>

    

  

  
    

        

            

                Class
                Description
            

            

                .btn
                Adds basic styling to any button
            

            

                .btn-primary
                Provides extra visual weight and identifies the primary action in a set of buttons
            

            

                .btn-xs
                Makes an extra small button
            

            

                .btn-sm
                Makes a small button
            

            

                .btn-md
                Makes a medium button
            

            

                .btn-lg
                Makes a large button
            

            

                .btn-block
                Makes a block-level button (spans the full width of the parent element)
