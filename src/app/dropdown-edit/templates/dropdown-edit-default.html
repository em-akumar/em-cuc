                Class/Properties
                Description
            

            

                .em-dropdown-edit
                This class is required. Indicates a default/standard dropdown
            

             

                defaultText
                Display default selected text on the dropdown
            

            

                onChange
                Expose OnChange method, It helps to change the items on the selection
            

            

                valueField
                The value which user wants to keep or store in the database
            

            

                textField
                Display text on the selection of dropdown
            

            

                defaultSize
                Define the size of drodown
            

            

                sortField
                Define the sort field name in drodown
            

            

                sortOrder
                Define the sort order asc/desc
