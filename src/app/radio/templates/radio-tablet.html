Radio Button Usage
<div class="radio">
          <input type="radio" id="radio22" name="optradio" value="option1">
          <label for="radio22">Default</label>
        </div>
        <div class="radio">
          <input type="radio" id="radio22" name="optradio1"  value="option1" checked>
          <label for="radio22">Checked</label>
        </div>
        <div class="radio">
          <input type="radio" id="radio22" name="optradio1"  value="option1">
          <label for="radio22">Pressed</label>
        </div>
        <div class="radio">
          <input type="radio" id="radio20" name="optradio" disabled>
          <label for="radio20">Disabled</label>
        </div>

    

  



  

    

      
        

          Property
          Description
        

        

          checked
          Adds a checked state of the radio button
        

        

          disabled
          Adds a disabled state of the radio button
