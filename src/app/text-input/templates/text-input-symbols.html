                    Class
                    Description
                

                

                    .em-txt-decimal-upto
                    Accepts only Integer number when .em-txt-decimal-upto="0"
                

                

                    Accepts only Floating point number when .em-txt-decimal-upto = "<number of digits after decimal point>"
                

                

                    Accepts only positive numbers when used with em-allow-negative="false"
                

                

                    .em-input-symbol-percent
                    Makes an percentage symbol on right hand side of text input field
                

                

                    .em-input-symbol-dollar
                    Makes an dollar symbol on left hand side of text input field
                

                

                    .em-input-symbol-months
                    Makes an Months text on right hand side of text input field
                

                

                    .em-input-symbol-percent-24
                    Makes an percentage symbol on right hand side of text input field with height 24px
                

                

                    .em-input-symbol-dollar-24
                    Makes an dollar symbol on left hand side of text input field with height 24px
                

                

                    .em-input-symbol-months
                    Makes an Months text on right hand side of text input field with height 24px
