Default Button Usage
<div class="demo-example" data-heading="Secondary text button states">
     <button type="button" class="btn btn-default">Default</button>
     <button type="button" class="btn btn-default">Hover</button>
     <button type="button" class="btn btn-default active">Active</button>
     <button type="button" class="btn btn-default disabled">Disabled</button>
</div>

    

  


    
    

        

            

                Class
                Description
            

            

                .btn-default
                Indicates a default/standard button
            

            

                .active
                Makes the default button appear pressed
            

            

                .disabled
                Makes the default button disabled
            

             

                icon button
                Please add HTML span tag inside button tag and use icon in the class to add icon inside the button
