Tertiary Button Usage
<div class="demo-example" data-heading="Tertiary text button states">
     <button type="button" class="btn em-btn-tertiary">Default</button>
     <button type="button" class="btn em-btn-tertiary">Hover</button>
     <button type="button" class="btn em-btn-tertiary active">Active</button>
     <button type="button" class="btn em-btn-tertiary disabled">Disabled</button>
</div>

    

  


    
    

        

            

                Class
                Description
            

            

                .em-btn-tertiary
                Indicates a tertiary button
            

            

                .active
                Makes the tertiary button appear pressed
            

            

                .disabled
                Makes the tertiary button disabled
            

             

                icon button
                Please add HTML span tag inside button tag and use icon in the class to add icon inside the button
