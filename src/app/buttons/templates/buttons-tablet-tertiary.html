Button Tertiary Usage

        <button type="button" class="btn btn-tertiary">tertiary</button>
      <button type="button" class="btn btn-tertiary active">Active</button>
      <button type="button" class="btn btn-tertiary disabled">Disabled</button>
        

    

  


   
    

        

            

                Class
                Description
            

            

                .btn-tertiary
                Indicates a tertiary/standard button
            

            

                .active
                Makes the tertiary button appear pressed
            

            

                .disabled
                Makes the tertiary button disabled
