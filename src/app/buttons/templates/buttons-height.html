Button Sizes Variable Height Usage
<div id="button-variable-height" class="panel-body">
    <div class="demo-example" data-heading="Button Sizes Variable Height">
      <button type="button" class="btn btn-primary btn-xs-var-height" title="xs-var-height">ok</button>
      <button type="button" class="btn btn-primary btn-sm-var-height" title="sm-var-height">ok</button>
      <button type="button" class="btn btn-primary btn-md-var-height" title="md-var-height">ok</button>
      <button type="button" class="btn btn-primary btn-lg-var-height" title="lg-var-height">ok</button>
    </div>
</div>

    

  


    
    

        

            

                Class
                Description
            

            

                .btn-xs-var-height
                Makes an extra small button with variable height
            

            

                .btn-sm-var-height
                Makes a small button with variable height
            

            

                .btn-md-var-height
                Makes a medium button with variable height
            

            

                .btn-lg-var-height
                Makes a large button with variable height
