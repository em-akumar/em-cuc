Button Sizes Variable Width Usage
  <div id="button-variable-width" class="panel-body">
    <div class="demo-example" data-heading="Button Sizes Variable Width">
      <button type="button" class="btn btn-primary btn-xs-var-width" title="xs-var-width">ok</button>
      <button type="button" class="btn btn-primary btn-sm-var-width" title="sm-var-width">ok</button>
      <button type="button" class="btn btn-primary btn-md-var-width" title="md-var-width">ok</button>
      <button type="button" class="btn btn-primary btn-lg-var-width" title="lg-var-width">ok</button><br/><br/>
      <button type="button" class="btn btn-primary btn-flexible-var-width" title="flexible-var-width">ok</button>
    </div>
</div>

    

  


    
    

        

            

                Class
                Description
            

            

                .btn-xs-var-width
                Makes an extra small button with variable width
            

            

                .btn-sm-var-width
                Makes a small button with variable width
            

            

                .btn-md-var-width
                Makes a medium button with variable width
            

            

                .btn-lg-var-width
                Makes a large button with variable width
            

            

                .btn-flexible-var-width
                Makes a flexible button with variable width
