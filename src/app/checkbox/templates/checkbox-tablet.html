Tablet Check Box

CheckBox Control Usage
 <div class="checkbox"> <input type="checkbox" name="default" id="default">
        <label>Default</label> </div> <div class="checkbox"> <input type="checkbox"
        id="checked" checked> <label>Checked</label> </div> <div class="checkbox">
        <input type="checkbox" id="Pressed"> <label>Pressed</label> </div> <div
        class="checkbox"> <input type="checkbox" id="Disabled" disabled> <label>Disabled</label>
        </div>
      

    

  



  
  

    

      
        

          Property
          Description
        

        

          checked
          Adds a checked state of the checkbox
        

        

          disabled
          Adds a disabled state of the checkbox
